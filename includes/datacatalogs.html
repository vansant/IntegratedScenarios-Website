Data Catalogs for Integrated Scenarios Products

The Integrated Scenarios datasets are hosted on the Northwest Knowledge Network(NKN). You can download the data from this server either directly using OPeNDAP or from the internet. There are two types of data files:

  • 'Aggregated': here are files for each scenario which have been aggregated over years (i.e. 1950-2005 and 2006-2100). These data are ideal to use for those users that want to subset the data either in space or time and are going to use either the OPeNDAP tools or the internet tools to do the subsetting.
  • 'Not Aggregated': here are files which are 10 to 20-years of data for each scenario. These data are ideal to download for those users that want to download the full domain of each dataset.
